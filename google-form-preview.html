Ini adalah PREVIEW visual Google Form Gunakan file google-form-apps-script.gs untuk membuat form nyata di Google, dan booking-form-connector.js untuk menghubungkannya ke RoomBook API.

🚀 Cara Setup (3 Langkah)

  1. Buat Google Form di forms.google.com dengan field sesuai preview di bawah
  2. Pasang Apps Script — buka form → ⋮ → Script editor → paste isi file google-form-apps-script.gs
  3. Set trigger — di Apps Script → Triggers → On form submit → jalankan fungsi onFormSubmit
